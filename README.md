## Usage

Run the executable from the command line and provide the required arguments:

```sh
./sprite-extractor --input_file=<path> --output_dir=<path>
```

### Example

```sh
./sprite-extractor --input_file=./input --output_dir=./output
```

### Arguments

| Argument              | Description                   |
| --------------------- | ----------------------------- |
| `--input_file=<path>` | Path to the input .lod file.  |
| `--output_dir=<path>` | Path to the output directory. |

Both arguments are required.
